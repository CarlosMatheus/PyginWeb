Balance

A simple and challenging arcade game made as a college project.

The development of Pygin started with Balance. Built on top of the Pygame framework, the Pygin engine was created as a way of making it easier to create more complex games using python.

The Pygin engine was highly inspired by the Unity engine architecture and the PyginWeb project was created as a way to spread this technology and save people time.
